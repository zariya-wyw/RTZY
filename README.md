# RTZY Shopify Theme

A premium Shopify theme featuring ultra-luxury glassmorphic product cards, image carousels, and responsive grid layouts.

## Quick Git Commands

### Push Changes to Remote Repository

```bash
# Check status
git status

# Add all changes
git add .

# Commit with message
git commit -m "Your commit message here"

# Push to main branch
git push origin main

# View recent commits
git log --oneline -5

# Specific example:
cd C:\Users\serat\OneDrive\Documents\Libraries\Zariya\RTZY
git add assets/zariya-styles.css assets/zariya-shopify.css snippets/product-card.liquid sections/featured-products.liquid
git commit -m "Your changes description"
git push origin main
```
